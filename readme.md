## 🌟 Cowork in Aktion

### Warum AionUi wählen?

AionUi ist eine kostenlose, lokale und Open-Source-Cowork-Anwendung, mit der du rund um die Uhr mit verschiedenen KI-Agenten zusammenarbeiten kannst. Die App bringt einen eigenen integrierten Agenten mit und kann zusätzlich externe Agenten wie Gemini-CLI, Claude Code, Codex, OpenCode, Qwen Code, Goose CLI, Auggie und andere in einer einheitlichen Oberfläche bündeln.

### ✨ Hauptmerkmale

- **Integrierter Agent ohne Setup**: Installation, Google-Anmeldung oder API-Schlüssel reichen aus – Dateizugriff, Websuche, Bildgenerierung und Tools sind sofort einsatzbereit
- **Multi-Agent-Betrieb**: Erkennt vorhandene CLI-Agenten automatisch und vereint sie in einer Oberfläche mit parallelen Sitzungen und zentralem MCP-Tool-Management  
- **Beliebige API-Keys**: Unterstützt Gemini, OpenAI, Anthropic, Ollama/LM Studio, NewAPI und mehr – immer mit vollem Cowork-Funktionsumfang
- **Automatisierung & Workflows**: Geplante Tasks per natürlicher Sprache und Cron für Reports, Datenaggregation, Dateiorganisation u.v.m. (24/7 unbeaufsichtigt)
- **Vorschau & Bearbeitung**: Für über 10 Formate (PDF, Office, Code, Markdown, Bilder, HTML) inkl. Versionierung und Live-Bearbeitung
- **Smarte Dateiverwaltung**: Batch-Rename, Sortierung, Zusammenführung, Excel-Analyse, Dokument- und Bildgenerierung durch KI-Agenten
- **Fernzugriff**: WebUI über LAN/Internet oder Chat-Integrationen wie Telegram, Lark, DingTalk, Slack (in Arbeit)
- **Mehrere parallele Chats**: Jede Sitzung mit eigenem Kontext und visueller Übersicht  
- **Vollständig anpassbar**: Oberfläche per eigenem CSS komplett konfigurierbar

## 🚀 Schnellstart

Wähle deine Plattform:

### macOS

Lade die `.dmg`-Datei herunter und installiere sie.

### Windows  

Lade die `.exe`-Datei herunter und führe sie aus.

### Linux

Lade die AppImage-Datei herunter und mache sie ausführbar:

```bash
chmod +x AionUi.AppImage
./AionUi.AppImage
```

### Mit Homebrew (macOS)

```bash
brew install aionui
```

## ⚙️ Konfiguration

Bei der ersten Nutzung wirst du durch eine einfache Oberfläche geleitet, in der du API-Keys einbindest und externe Agenten konfigurierst.

### Unterstützte API-Anbieter

- **Google Gemini**
- **OpenAI** (GPT-4, GPT-3.5, etc.)
- **Anthropic Claude**
- **Ollama** (lokal)
- **LM Studio** (lokal)
- **NewAPI**

## 📚 Anleitungen

### Dateien hochladen

Ziehe Dateien direkt in die Chatoberfläche oder verwende den Upload-Button.

### Web durchsuchen  

Lass Agenten das Web durch einfache Befehle durchsuchen und Informationen sammeln.

### Code bearbeiten

Code oder Konfigurationsdateien können direkt in der Preview bearbeitet und gespeichert werden.

### Automatisierte Tasks planen

Erstelle Cron-Jobs oder zeitgesteuerte Tasks in natürlicher Sprache – AionUi kümmert sich um die Ausführung.
